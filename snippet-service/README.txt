Чтобы поднять кассандру надо в папке сервиса написать:
docker-compose up
Насколько помню, в этот момент должен быть запущен докер.
Далее заходите через cmd и пишите
docker ps
В списке находите кассандру, копируете айди контейнера
Вставляете в команду:
docker exec -it <айди> bash
Попадаете в командную строку контейнера
cqlsh
Вы в кассандре! Далее - команды по созданию таблицы и пространства. Они в файлике database-creation.txt

ВНИМАНИЕ! Айди сниппета в базе данных соответствует айди поста, к которому он прикреплён. Так что отправляйте запросы с умом
